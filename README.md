Sistema completo de punto de venta (POS) en un solo archivo. Login, inventario, carrito, dashboard y modo oscuro — listo para usar sin instalar nada.

## ✨ Live Demo

👉 **[Ver demo en vivo](https://abrahamramoskd.github.io/ventapro)**
## 🚀 Funcionalidades

- 🔐 **Login con credenciales** — acceso protegido al sistema
- 📦 **Gestión de inventario** — agrega, edita y elimina productos
- 🛒 **Carrito de compras** — flujo completo de venta
- 📊 **Dashboard con métricas** — resumen de ventas y actividad
- 🏆 **Top productos** — ranking de los más vendidos
- 🌙 **Modo oscuro / claro** — toggle instantáneo

## 🛠️ Tecnologías

- HTML5 semántico
- CSS puro (variables, flexbox, grid)
- Vanilla JS (sin dependencias)

## 📁 Estructura

```
ventapro/
└── service.html   ← Todo en un solo archivo
```

## 🚀 Uso

Solo abre `service.html` en tu navegador — no requiere servidor ni instalación.
